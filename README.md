# KNameCards
名刺管理アプリケーション（Electron)
Name Card Manager (Electron)

## 機能 (Functions)
### 表示 (Viewer Mode)
- 名刺情報を表示する (Show name card list)
- 表示項目を選択する (Select name card items)
- 名刺を検索する (Search name cards)

### 入力
- 名刺情報を入力する (Input name card)

## 操作
### 表示<->入力切替
Comand+I

### 表示項目選択
Command+D

### 検索
Command+F
